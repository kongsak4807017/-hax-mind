## Medium
HAX-Mind is a Telegram-first local AI operating system for Windows that combines project/task orchestration, hybrid memory intelligence, guarded real code apply with rollback, and local production operations such as health checks, recovery, alerts, backup, and restore drill workflows.

HAX-Mind is a local AI operating system for Windows designed to be operated through Telegram. It includes project and mission management, proposal/approval workflows, hybrid recall and vector-based memory intelligence, guarded real code execution with backup/diff/rollback, and a full local production toolchain with health snapshots, recovery, alerts, backup bundles, restore drills, operator dashboard, and runbooks.
